# FluidCAD

**FluidCAD** es un software libre y de código abierto (licencia **GPL-3.0**) que fusiona y mejora las capacidades de **CADESIMU** y **FluidSIM** en una única aplicación moderna, potente y completamente gratuita.

### Objetivo del proyecto
Desarrollar una herramienta unificada, comunitaria y sin restricciones de licencia para el diseño, simulación y enseñanza de sistemas de:

- Neumática  
- Hidráulica  
- Electroneumática  
- Electrohidráulica  

FluidCAD busca combinar la intuitividad y calidad didáctica de FluidSIM con la filosofía abierta y la flexibilidad de CADESIMU, eliminando las limitaciones propietarias y permitiendo que estudiantes, profesores, técnicos, ingenieros y entusiastas de todo el mundo contribuyan al desarrollo.

### Características principales (en desarrollo / planificadas)

- Editor gráfico intuitivo con drag-and-drop para crear circuitos neumáticos, hidráulicos y eléctricos  
- Simulación en tiempo real del comportamiento dinámico (presión, caudal, desplazamientos, fuerzas, tiempos de respuesta…)  
- Amplia librería de componentes: válvulas, cilindros, motores, acumuladores, sensores, relés, contactores, temporizadores, contadores, PLCs básicos, etc.  
- Componentes extensibles y actualizables por la comunidad  
- Modo educativo con animaciones claras, leyendas explicativas y visualización paso a paso  
- Exportación de diagramas y simulaciones a PDF, PNG, SVG y (próximamente) vídeo  
- Multiplataforma: Windows, Linux y macOS  
- 100 % gratuito, sin versiones de pago ni restricciones educativas  
- Código abierto bajo **GNU GPL-3.0** — cualquiera puede modificarlo, traducirlo, mejorar la simulación o añadir nuevas funcionalidades

### Estado actual
El proyecto está en **fase inicial**:

- Análisis y migración del núcleo de simulación de CADESIMU y FluidSIM  
- Definición de la arquitectura modular para facilitar contribuciones  
- Rediseño de la interfaz de usuario (más moderna y responsive)  
- Creación de una librería de componentes común y bien documentada
