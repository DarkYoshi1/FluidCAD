# FluidCAD

**FluidCAD** es un software libre y de código abierto (licencia GPL-3.0) que unifica y extiende las mejores características de **CADESIMU** y **FluidSIM** en una sola aplicación moderna y potente.

### Objetivo principal
Crear una herramienta única, gratuita y comunitaria para el diseño, simulación y enseñanza de sistemas de:

- Neumática  
- Hidráulica  
- Electroneumática  
- Electrohidráulica  

FluidCAD nace con la visión de combinar la facilidad de uso y la interfaz intuitiva de FluidSIM con la filosofía abierta y la flexibilidad de desarrollo de CADESIMU, eliminando barreras de licencia propietaria y permitiendo que estudiantes, docentes, técnicos e ingenieros de todo el mundo colaboren y mejoren el programa.

### Características clave (en desarrollo / planificadas)

- Editor gráfico drag-and-drop para circuitos neumáticos, hidráulicos y eléctricos  
- Simulación en tiempo real de comportamiento dinámico (presión, caudal, velocidad, fuerzas…)  
- Soporte completo para válvulas, cilindros, motores, sensores, relés, contactores, temporizadores, contadores…  
- Librerías de componentes extensibles y actualizables por la comunidad  
- Modo didáctico con animaciones claras, leyendas y explicación paso a paso  
- Exportación a PDF, imágenes y (próximamente) vídeos de simulación  
- Multiplataforma: Windows, Linux y macOS  
- Totalmente gratuito, sin licencias comerciales ni restricciones educativas  
- Código abierto bajo **GPL-3.0** → ¡cualquiera puede modificarlo, traducirlo o añadir funcionalidades!

### Estado actual del proyecto
Proyecto en fase inicial de desarrollo.  
Estamos trabajando en:

- Migrar y unificar el núcleo de simulación de CADESIMU y FluidSIM  
- Rediseñar una interfaz moderna y responsive  
- Crear una base de componentes común y extensible  
- Definir la arquitectura para facilitar contribuciones
